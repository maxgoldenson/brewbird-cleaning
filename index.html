Cleaning Tablet Procedure

Step-by-step guide for the BrewBird brewer

You will need: a Cleaning Tablet Pod, a Rinse Pod, a Dry Pod, a Cafiza cleaning tablet, the tablet wand, and a pitcher (1000 mL capacity or higher). Tap a photo to view it full size.

1

Insert Cleaning Tablet Pod

  1. Insert the Cleaning Tablet Pod.
  2. Touch the screen to start the cycle.
2

Remove Tray

  1. When prompted, remove the tray.
  2. Set the tray aside.
⚠️The tray has moving parts — hold it securely as shown.
3

Place Cafiza Tablet Into Wand

  1. Place a Cafiza cleaning tablet into the wand as shown.
4

Deploy Tablet

  1. Insert the wand into the chute — the tablet should face down.
  2. Verify the tablet was deployed: listen carefully for the tablet to fall.
⚠️Verify the tablet did not fall into the trash area or onto the floor.
5

Run Cleaning Tablet Cycle

  1. Reinsert the tray — the action should be smooth.
  2. Place a pitcher (1000 mL capacity or higher) on the tray.
  3. Touch the screen to start the cleaning cycle.
6

Run Rinse Cycle

  1. After the cleaning cycle is complete, remove the Cleaning Tablet Pod.
  2. Insert the Rinse Pod.
  3. Touch the screen to start the rinse cycle.
7

Run Blank Cycle

  1. After the rinse cycle is complete, remove the Rinse Pod.
  2. Insert the Dry Pod.
  3. Touch the screen to start the blank cycle.
8

Cleanup

✅

Cleaning Tablet Procedure Complete

The brewer is clean and ready for use.
