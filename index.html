Cleaning Tablet Procedure

Step-by-step guide for the BrewBird brewer

0 / 8 steps

Before you start — gather these items

Tap any photo to enlarge it. Tap Mark step done as you go — progress is saved on this phone, so you can lock the screen between cycles. Use Reset to start a new cleaning.

1

Insert Cleaning Tablet Pod

  1. Insert the Cleaning Tablet Pod.
  2. Touch the screen to start the cycle.
2

Remove Tray

  1. When prompted on the screen, remove the tray.
  2. Set the tray aside.
⚠️The tray has moving parts — hold it securely as shown in the photo marked !.
3

Place Cafiza Tablet Into Wand

  1. Place a Cafiza cleaning tablet into the wand as shown.
4

Deploy Tablet

  1. Insert the wand into the chute — the tablet should face down.
  2. Push until fully inserted and you feel a click.
  3. Verify the tablet was deployed: listen carefully for the tablet to fall.
⚠️Verify the tablet did not fall into the trash area or onto the floor.
5

Run Cleaning Tablet Cycle

  1. Reinsert the tray — the action should be smooth.
  2. Place the pitcher (1000 mL capacity or higher) on the tray.
  3. Touch the screen to start the cleaning cycle.
6

Run Rinse Cycle

⏱️Wait for the cleaning cycle to finish before starting this step.
  1. Remove the Cleaning Tablet Pod.
  2. Insert the Rinse Pod.
  3. Touch the screen to start the rinse cycle.
7

Run Blank Cycle

⏱️Wait for the rinse cycle to finish before starting this step.
  1. Remove the Rinse Pod.
  2. Insert the Dry Pod.
  3. Touch the screen to start the blank cycle.
8

Cleanup

⏱️Wait for the blank cycle to finish before starting cleanup.
✅

Cleaning Tablet Procedure Complete

The brewer is clean and ready for use.
